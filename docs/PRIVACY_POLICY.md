# 개인정보 처리방침

**시행일: 2026년 1월 13일**

Tori Wallet(이하 "앱")은 사용자의 개인정보를 중요시하며, 개인정보보호법 등 관련 법령을 준수합니다.

## 1. 수집하는 개인정보

### 자동 수집 정보

- **기기 정보**: 기기 모델, 운영체제 버전
- **앱 사용 정보**: 앱 충돌 로그, 성능 데이터

### 수집하지 않는 정보

- **니모닉(복구 구문)**: 사용자 기기에만 암호화되어 저장되며, 외부로 전송되지 않습니다.
- **개인키**: 사용자 기기의 보안 저장소에만 저장됩니다.
- **트랜잭션 내역**: 블록체인에 공개적으로 기록되며, 앱 서버에는 저장되지 않습니다.

## 2. 개인정보의 이용 목적

- 앱 서비스 제공 및 개선
- 오류 분석 및 서비스 안정성 향상
- 사용자 문의 대응

## 3. 개인정보의 보관 및 파기

- **보관 기간**: 앱 삭제 시까지
- **파기 방법**: 앱 삭제 시 기기 내 모든 데이터가 자동으로 삭제됩니다.

## 4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외로 합니다:

- 법령에 의거하거나 수사기관의 요청이 있는 경우
- 사용자가 사전에 동의한 경우

## 5. 제3자 서비스

앱은 다음 제3자 서비스를 이용할 수 있습니다:

### 블록체인 네트워크

- Ethereum, Polygon, Arbitrum, Optimism, Base 등 퍼블릭 블록체인
- 트랜잭션은 해당 블록체인에 공개적으로 기록됩니다.

### WalletConnect

- dApp 연결 서비스
- 연결 시 지갑 주소가 dApp에 공유됩니다.

### 외부 API

- CoinGecko (가격 정보)
- 블록체인 RPC 프로바이더

## 6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

- **데이터 삭제**: 앱 내 "지갑 초기화" 기능 또는 앱 삭제로 모든 데이터 삭제
- **데이터 조회**: 저장된 데이터는 모두 사용자 기기에 있으며, 앱 내에서 확인 가능

## 7. 보안

- 니모닉 및 개인키는 iOS Keychain / Android Keystore에 암호화되어 저장
- PIN 및 생체인증을 통한 앱 접근 제어
- 모든 네트워크 통신은 HTTPS 암호화

## 8. 아동의 개인정보

본 앱은 만 14세 미만의 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.
